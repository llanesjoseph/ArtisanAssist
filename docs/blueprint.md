# **App Name**: CraftCanvas

## Core Features:

- User Authentication: User authentication via Google sign-in to personalize the app experience.
- Dashboard: Dashboard providing an overview of ongoing projects, encouraging the start of new ones, and managing existing ones.
- Project Creation Wizard: A modal wizard that simplifies the process of entering new project details, accommodating project names, types, and specifics.
- Project List Display: Showcasing each project with key details at a glance and offering direct actions to view or delete projects.
- Project Details Management: A centralized space to manage every project detail, from notes and dimensions to tool settings, visual aids, and progress tracking.
- Woodworking Cut Plan Tool: Facilitating the meticulous planning and optimized material use for woodworking through the cut plan tool, supporting efficiency in material processing.
- Inspiration and progress logging: Capture links and images for reference, while logging your progress through words and photos.

## Style Guidelines:

- Primary color: HSL(39, 75%, 50%) / RGB(#F29727), a warm, inviting color that promotes creativity.
- Background color: HSL(39, 20%, 90%) / RGB(#F2EAE0), a very light, desaturated shade that provides a soft, non-distracting backdrop.
- Accent color: HSL(9, 75%, 45%) / RGB(#D95A25), a contrasting analogous color that provides visual interest for secondary interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, suitable for headlines or body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, line-based icons for categories and actions.
- Subtle transitions and animations for feedback.
